以后一段时间内规划的笔记标准工作流：

<Steps>
1. 明确自己要学习哪一门课程
2. 通过 <Icon name='github' class='inline' /> github 了解这门课相关的网课、课件、Lab、作业以及其它相关资料（学院课程已经 fetch 到本地）
3. 得到所有课程群内的资源后第一时间进行收集整理
4. 在写文档之前，先对课程群里已经发了的资料了解清楚，尽量避免没有理解的情况下就总结文档。
5. 学习过课件后总结文档。需要在当周内总结完当周内所有的文档
</Steps>
